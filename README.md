# Secure Video Upload, Sensitivity Analysis & Streaming Platform

A production-grade full‑stack application for secure video uploads, real‑time
processing updates, sensitivity classification, and HTTP range streaming.

## Features
- JWT Authentication & RBAC
- Multi-tenant user isolation
- Real-time processing via Socket.io
- Secure video streaming
- Clean, scalable architecture

## Tech Stack
Backend: Node.js, Express, MongoDB, Socket.io, FFmpeg  
Frontend: React, Vite

## Setup
1. Copy `.env.example` → `.env`
2. Start backend: `npm run dev`
3. Start frontend: `npm run dev`
